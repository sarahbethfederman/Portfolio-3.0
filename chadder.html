Chadder

Website Reboot

Project Description:

Chadder is an RIT-based startup. They created an end-to-end encrypted private messaging mobile application. In need of a site redesign, they approached me with the project. I designed the website for my GUI course project and continued on to develop the site, until I helped officially launch the app at CES 2015 in Las Vegas. Utilizing a responsive approach, the site's main function is to increase app downloads. Currently featured on Product Hunt!

Time: 3 weeks UI/UX design Concept/Strategy HTML5 jQuery SCSS

Development Cycle

Front-end Development

This was a bit of a sprint, since we only had about a week to prepare in between final exams and leaving for CES. Because of this, I had to nix some of the extra effects I'd intended, like parallax scrolling.

Comp Matching

I used Adobe Extract to quickly get critical information about colors, dimensions, etc.

Modals

With the help of CSS animations and some jQuery, I created custom modals for non-critical information

PX to REMs

All pixel values were converted to rems by use of a sass function, from the Bourbon sass framework

Mobile Responsive

With the help of some media query adjustments, the mobile version came to life

Final Thoughts

This was a fun project because I learned a lot about working with startups. I took on more roles than just designer/developer, and got to partake in some strategy. At CES, I spent a ton of time pitching to potential users at our booth.

If I could do it again, I would give myself some more time to create some more enhancements. I might also forgo using rems as a unit relative to the viewport height(vh). While it made many things responsive with little effort, it also overrode some things which I had to countercorrect, especially with type.
